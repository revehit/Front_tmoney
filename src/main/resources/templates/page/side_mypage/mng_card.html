15자리 카드번호 입력 시 아래 화살표를 눌러 15자리를 선택 후 카드번호를 등록해주세요

- - -

교통카드 기능이 없는 카드는 등록이 불가능합니다.

신한Amex카드(15자리)는 번호 입력 시 숫자가 하나 더 생성되어 16자리로 저장 됩니다.

월1회 변경이란 월중 하루만 해당 카드로 변경 가능합니다.(사용자가 카드 변경일을 선택하여 등록)

변경일에는 제한 없이 카드번호 변경이 가능합니다.(최종 변경 카드번호만 해당일에 등록됨)

추가 변경 건은 고객센터로 문의하시길 바랍니다.

잘못된 카드번호를 등록할 경우 지원금을 신청할 수 없습니다.

번호 등록 시 반드시 카드번호를 한번 더 확인하시길 바랍니다.
