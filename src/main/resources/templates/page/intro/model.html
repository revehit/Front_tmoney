교통복지 플랫폼 모델별

All-in-One형

홈페이지 구축, 데이터 정산, 고객센터 운영 모두 포함된 통합형 서비스 모델

지자체 혜택

  • 지자체의 행정부담을 최소화하고 사업 전 과정의 효율적인 관리 가능
  • 신청·지급·민원응대 전 과정 통합 관리 가능
  • 지자체 행정 부담 최소화 혜택 제공

Standard-A형

홈페이지 구축 및 데이터 정산 서비스가 포함된 표준형 모델

지자체 혜택

  • 홈페이지 구축과 DATA 정산 서비스 제공
  • 신청·지원금 지급 등 일반 서비스 가능
  • 기존 민원응대 시스템 보유 지자체에 적합

Standard-B형

고객센터 운영 및 데이터 정산 서비스가 포함된 표준형 모델

지자체 혜택

  • 홈페이지 구축과 DATA 정산 서비스 제공
  • 신청·지원금 지급 등 일반 서비스 가능
  • 기존 민원응대 시스템 보유 지자체에 적합

Lite형

정책에 필요한 데이터만 제공하는 경량형 모델

지자체 혜택

  • 정책에 필요한 DATA 제공(또는 정산)
  • 자체 시스템·홈페이지 운영 중인 지자체에 적합
  • DATA 정산만 필요한 경우 선택 가능
